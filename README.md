## dfcli
Commandline tool for DevFlow (https://github.com/fiverr/devflow)

#Configuration:
  * Save/Build dfcli executable.
  * Add DFCLI_END_POINT environment varaiable to .bash_profile ```export DFCLI_END_POINT="localhost:3000"```
  * Add dfcli location to $PATH ```export PATH=$PATH:/path/to/dfcli```

run ```dfcli init```
